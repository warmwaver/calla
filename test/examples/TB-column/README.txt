使用方法：
1、使用文本编辑器打开文件“input.json”，按格式输入数据。支持多个数据同时计算，文件命名规则为“input_柱名.json”。
输入数据含义如下：
{
    "b":1300, # 宽度(mm)
    "h":1600, # 高度(mm)
    "concrete":"C40", # 混凝土等级
    "rebar":"HRB400", # 钢筋等级
    "d":28, # 钢筋直径(mm)
    "As":16084, # 受拉钢筋面积(mm)
    "As_":16084, # 受压钢筋面积(mm)
    "a":110, # 受拉钢筋距截面边距离(mm)
    "a_":110, # 受压钢筋距截面边距离(mm)
    "l0":5, # 柱计算长度(m)
    "n":10, # 钢筋的弹性模量和混凝土的变形模量之比
    "N":[
        8198.88,
        7633.59,
        6065.35
    ], # 主力、主力+附加力、主力+地震力作用下的轴力(kN)
    "M":[
        1024.85,
        2621.61,
        6437.93
    ], # 主力、主力+附加力、主力+地震力作用下的弯矩(kN・m)
    "M1":107.2, # 活载作用下的弯矩(kN・m)
    "M2":917.7, # 恒载作用下的弯矩(kN・m)
    "V":[
        319.96,
        612.33,
        1302.53
    ] # 主力、主力+附加力、主力+地震力作用下的剪力(kN)
}
2、运行“TB.exe”，计算结束后查看“result_*.html”文件。
